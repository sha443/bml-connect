# BMLConnect
`sha443/bml-connect` is a laravel package for Bank of Maldives Connect API v2.0.


## Pre-requisites

- Requires PHP 7.3 or higher
- Laravel 5.5 or higher 

## Installation
`composer require sha443/bml-connect`

Don't forget to run
`composer dump-autoload`
